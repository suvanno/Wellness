== Description ==

A Responsive WordPress theme for Health, Fitness & Wellness business.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= How do I set up front page like the demo? =

Follow these steps below:

1. Install and activate the Jetpack plugin.
2. Create new page and assign template page attributes to `Front Page`
3. Go to Settings -> Reading -> Front page displays - A static page -> Set the page you created in step 2 as Front Page
4. Go to Customizer -> Featured Content from the WP Admin Dashboard. Enter a tag that you would like to use on
Featured Content posts. For example, you could use the tag 'featured'.
5. Go to Customizer -> Theme Options -> Featured Page Section from the WP Admin Dashboard. Select pages you would like to
display in featured page section.
6. Go to Customizer -> Theme Options -> Grid Page Section from the WP Admin Dashboard. Select a page which has sub-pages
then all sub-page will be display in a grid layout.
7. Go to Customizer -> Testimonials Section and check Display Testimonials, then four random testimonials will be display
on the Front Page template. You can add new Testimonial by navigate to Testimonial menu from WP Admin Dashboard.
8. Go to Customizer -> Latest News Section and check Display latest news section then four latest posts will be display
in a grid layout on the Front Page template.

= How do I set sidebar position to left? =

Go to Customizer -> Theme Options -> General Settings and choose the `left` sidebar position.

= Does this theme support any plugins? =

Wellness includes support for Infinite Scroll in Jetpack.

= How do I set up a full width template? =

Create or edit a page, assign it the Full-Width Page template and click Publish.

= How do I set up a grid page template? =

1. Create or edit a page, and assign it to the Grid Page Template.
2. The Grid Page Template will display all sub-pages of the current page in a three-column grid layout.

= How do I set up a Testimonials page? =

All testimonials are displayed in an automatically generated testimonial archive page.
The testimonial archive page can be found on your site at http://yoursite.com/testimonial/ (replace http://yoursite.com/ with your own site URL).
You can add the testimonial archive to a Custom Menu, using a custom link.
You can change the Testimonial page title, content, featured image by going to Customizer -> Testimonials from the WP Admin Dashboard.
